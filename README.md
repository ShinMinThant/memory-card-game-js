# 🧠 Memory Game

A fun and interactive Memory Card Game built with **HTML**, **CSS**, and **Vanilla JavaScript**. Flip cards, find matching pairs, and complete the game with the fewest moves in the shortest time!

## ✨ Features

* 🎴 Flip memory cards
* 🔀 Randomly shuffled cards every game
* ✅ Match detection
* ❌ Automatically flip unmatched cards back
* 📊 Move counter
* ⏱️ Timer
* 🎉 Win message with game statistics
* 🔄 Restart game
* 📱 Responsive layout

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)

---

## 🎮 How to Play

1. Click any card to reveal it.
2. Click a second card.
3. If both cards match, they remain visible.
4. If they don't match, they flip back after a short delay.
5. Match all pairs to win the game.
6. Try to finish with the fewest moves and fastest time!
